## 🚀 Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build tools required — pure HTML/CSS/JS

### Deploy to GitHub Pages
This repo is already configured for GitHub Pages. Just push changes to the `main` branch.

## 📝 Development

### Adding Pages
1. Create a new `.html` file (e.g., `calendar.html`)
2. Keep the same `<nav>` structure for consistency
3. Use the same CSS variables from the root
4. Test on mobile and desktop before pushing

### Updating Navigation
The navigation is in the `<nav>` element at the top of each page. Keep it synchronized across all pages.

## 📱 Responsive Breakpoints

- **Desktop**: 1000px max-width centered layout
- **Tablet**: Full width with adjusted spacing
- **Mobile**: Single column, hamburger nav
